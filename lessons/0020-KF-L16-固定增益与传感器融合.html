固定增益估计器与传感器融合

🎯 本讲目标

1. 回顾：\$P_{k|k}\$ 收敛与稳态

上节课我们得出结论：当 \$(A, C)\$ 可观且 \$(A, \\sqrt{Q})\$ 能控时，\$P_{k|k}\$ 收敛到稳态值 \$\\bar P\$：

\$$ $\\lim_{k \\to \\infty} P_{k|k} = \\bar P$ \$$

随之而来的是 Kalman 增益的收敛：

\$$ $\\lim_{k \\to \\infty} K_{k} = K^* = \\bar P C' [C \\bar P C' + R]^{-1}$ \$$

💡 为什么收敛？

直观上，每个新测量都提供一定量的"信息"，而过程噪声又带来不确定性的增加。当信息收益 = 信息损失时，系统达到动态平衡——不确定性不再变化，增益也固定在最优值上。

2. 固定增益估计器

收敛后，KF 变成一个线性时不变系统（LTI system）：

\$$ $\\boxed{\\hat x_{k|k} = A \\hat x_{k-1|k-1} + K^* [y_{k} - C A \\hat x_{k-1|k-1}]}$ \$$

或者等价地写为预测形式：

\$$ $\\boxed{\\hat x_{k+1|k} = A \\hat x_{k|k-1} + K^* [y_{k} - C \\hat x_{k|k-1}]}$ \$$

固定增益的求解

\$K^*\$ 可以通过求解 代数 Riccati 方程（ARE）得到：

\$$ $\\bar P = A \\bar P A' - A \\bar P C' [C \\bar P C' + R]^{-1} C \\bar P A' + Q$ \$$

ARE 可以离线求解（例如用迭代法或直接数值解法），得到 \$\\bar P\$ 后再计算 \$K^* = \\bar P C' [C \\bar P C' + R]^{-1}\$。

🧠 关键认识

固定增益估计器是 KF 的"稳态版本"。它牺牲了初始阶段的时变最优性，换来了极大的计算简化。对于长时间运行的系统（如卫星的轨道确定），这是一个非常划算的交换。

3. 固定增益的工程实现优势

⬇️ 计算量显著降低

每一步只需：

  1. 矩阵-向量乘法 \$A \\hat x\$ 或 \$C A \\hat x\$
  2. 向量加减
  3. 常数矩阵-向量乘法 \$K^*\$

不再需要在线矩阵求逆，乘法次数从 O(n³) 降为 O(n²)。

💾 存储空间降为 O(1)

只需要存储：

  • 一个常数增益矩阵 \$K^*\$（大小 n×m）
  • 系统矩阵 \$A\$（已知）
  • 当前状态估计 \$\\hat x\$

不需要存储整个增益序列或协方差矩阵序列。

🔧 实现简单

代码量大幅减少，调试和维护成本低。对于 FPGA 或微控制器实现特别有吸引力。

📊 性能可预期

一旦 \$K^*\$ 确定，估计器的动态性能（收敛速度、稳态误差）可以通过经典的控制理论工具（极点分析等）进行分析。

⚠️ 权衡

固定增益估计器在系统参数不变时表现良好。但如果 \$Q\$、\$R\$ 或系统模型发生显著变化，\$K^*\$ 就不再是最优的。此时需要重新求解 ARE 或切换回时变 KF。

4. 信息形式 Kalman Filter

标准 KF 使用协方差 \$P\$ 和状态 \$\\hat x\$ 进行更新。有时候用它们的逆（信息矩阵 \$\\Omega = P^{-1}\$ 和信息向量 \$\\xi = P^{-1} \\hat x\$）更加方便。

从标准形式到信息形式

回顾后验协方差的更新：

\$$ P_{k|k}^{-1} = P_{k|k-1}^{-1} + C' R^{-1} C \$$

以及状态更新：

\$$ $P_{k|k}^{-1} \\hat x_{k|k} = P_{k|k-1}^{-1} \\hat x_{k|k-1} + C' R^{-1} y_{k}$ \$$

定义信息形式的变量：

信息矩阵

\$$ $\\Omega_{k|k} = P_{k|k}^{-1}$ \$$ \$$ $\\Omega_{k|k-1} = P_{k|k-1}^{-1}$ \$$

信息向量

\$$ $\\xi_{k|k} = P_{k|k}^{-1} \\hat x_{k|k}$ \$$ \$$ $\\xi_{k|k-1} = P_{k|k-1}^{-1} \\hat x_{k|k-1}$ \$$

信息形式的更新规则

\$$ $\\boxed{\\Omega_{k|k} = \\Omega_{k|k-1} + C' R^{-1} C}$ \$$ \$$ \\boxed{\\xi_{k|k} = \\xi_{k|k-1} + C' R^{-1} y_k} \$$

💡 解读

信息形式的测量更新就是简单的加法！每次测量到来，就把 \$C' R^{-1} C\$（信息贡献）加到信息矩阵上，把 \$C' R^{-1} y_{k}\$（加权测量）加到信息向量上。这种"累加"的结构非常适合多传感器融合。

5. 传感器融合与即插即用

问题设定

假设有 \$n\$ 个传感器，第 \$i\$ 个传感器的测量模型为：

\$$ y_{k}^i = C_{i} x_{k} + v_{k}^i, \\quad v_{k}^i \\sim N(0, R_{i}) \$$

堆叠法（标准形式）

将所有测量堆叠成一个向量，然后套用标准 KF：

\$$ y_{k} = \\begin{bmatrix} y_{k}^1 \\\\ y_{k}^2 \\\\ \\vdots \\\\ y_{k}^n \\end{bmatrix} = C x_{k} + v_{k}, \\quad C = \\begin{bmatrix} C_{1} \\\\ C_{2} \\\\ \\vdots \\\\ C_{n} \\end{bmatrix}, $\\quad v_{k} \\sim N(0, \\text{diag}(R_{1}, \\dots, R_{n}))$ \$$

这种方法需要每次拼装大矩阵，不够灵活。

信息形式（即插即用）

用信息形式，每个传感器的贡献独立累加：

\$$ $\\boxed{\\Omega_{k|k} = \\Omega_{k|k-1} + \\sum_{i=1}^{n} C_{i}' R_{i}^{-1} C_i}$ \$$ \$$ $\\boxed{\\xi_{k|k} = \\xi_{k|k-1} + \\sum_{i=1}^{n} C_{i}' R_{i}^{-1} y_{k}^i}$ \$$

🧩 模块化

每个传感器的贡献独立计算，互不干扰。新增传感器只需增加对应的 \$C_{i}' R_{i}^{-1} C_{i}\$ 项。

🔌 即插即用

某个传感器的数据来了就加进去，没来就不加。天然支持异步传感器和可变采样率。

🛡️ 容错性强

某个传感器失效时，只需移除其贡献项。不需要重新设计整个融合逻辑。

🔗 实际应用场景

自动驾驶汽车有摄像头、激光雷达、IMU、GPS 等传感器，每个传感器可能以不同频率运行、随时掉线或信号延迟。信息形式的 KF 天然支持这种动态配置的融合需求——每个传感器来了就"加"进去，走了就"减"出来。

6. 从几何到概率的过渡

到目前为止，我们一直使用几何方法（投影定理 + 新息正交性）来推导 KF。这种方法有两个局限：

  1. 我们得到的只是线性最优估计（LMMSE）——能否保证这是全局最优？
  2. 几何方法没有显式使用分布信息——能否利用高斯分布的特殊性质？

🤔 开放问题

在线性高斯系统下，LMMSE 是否等于 MMSE（最小均方误差估计，即条件均值 \$E[x_{k} | y_{0}, \\dots, y_{k}]\$）？如果等于，则说明 KF 的线性估计实际上是最优的——没有任何非线性方法能做得更好。

要回答这个问题，我们需要从概率的角度重新审视。这就是下一讲的内容——用条件高斯分布推导 MMSE 估计，并证明在线性高斯系统中 MMSE = LMMSE。同时还会从优化法的角度，将 Kalman 增益的求解转化为二次型最小化问题。

7. 本讲小结

本讲完成了 KF L16 的后半部分内容。总结核心要点：

概念 核心公式 工程意义
固定增益估计器 \$\\hat x_{k+1|k} = A \\hat x_{k|k-1} + K^* [y_{k} - C \\hat x_{k|k-1}]\$ O(1) 存储、O(n²) 计算
信息形式 KF \$\\Omega_{k|k} = \\Omega_{k|k-1} + C' R^{-1} C\$ 加法更新，便于融合
传感器融合 \$\\Omega = \\Omega_{0} + \\sum C_{i}' R_{i}^{-1} C_{i}\$ 即插即用、模块化、容错

问题 1

固定增益估计器相比时变 KF 有什么优势和劣势？

点击查看答案

优势：计算量小（O(n²) 而非 O(n³)）、存储开销小、实现简单。劣势：在初始阶段不是最优的（因为增益固定不能适应不确定性较大的初期）；系统参数变化时需要重新求解 ARE。

问题 3

信息形式的测量更新与标准形式的测量更新本质上是等价的吗？

点击查看答案

是的。信息形式 \$P_{k|k}^{-1} = P_{k|k-1}^{-1} + C' R^{-1} C\$ 可以通过矩阵求逆引理（SMW 公式）推导出标准形式 \$P_{k|k} = P_{k|k-1} - P_{k|k-1} C' [C P_{k|k-1} C' + R]^{-1} C P_{k|k-1}\$。两者是完全等价的，只是表达形式不同。
