1. 【课堂例题】偏硬币的 ML 推导

硬币正面概率为 \(\theta\)。第 \(i\) 次投掷：

\[ X_i = \begin{cases} 1, & \text{正面} \\ 0, & \text{反面} \end{cases} \]

单次观测概率：

\[ p(x_i; \theta) = \theta^{x_i} (1-\theta)^{1-x_i} \]

🔍 为什么这个式子成立？ 如果 \(x_i=1\)：\(\theta^1(1-\theta)^0 = \theta\)；如果 \(x_i=0\)：\(\theta^0(1-\theta)^1 = 1-\theta\)。一个式子覆盖两种情况。

2. 联合似然

假设 \(X_i\) 独立同分布：

\[ p(x; \theta) = \prod_{i=1}^n \theta^{x_i} (1-\theta)^{1-x_i} \]

取 log：

\[ \ell(\theta) = \sum_{i=1}^n x_i \log \theta + \sum_{i=1}^n (1-x_i) \log(1-\theta) \]

求导：

\[ \frac{d\ell}{d\theta} = \frac{1}{\theta} \sum_{i=1}^n x_i - \frac{1}{1-\theta} \sum_{i=1}^n (1-x_i) \]

令导数为 0：

\[ \frac{1}{\theta} \sum x_i = \frac{1}{1-\theta} \sum (1-x_i) \]

化简得到：

\[ \boxed{\hat\theta_{\text{ML}} = \frac1n \sum_{i=1}^n x_i} \]

💡 通俗解释： 估计硬币正面概率，最合理的答案就是历史上正面出现的比例。

3. 大数定律带来的收敛

如果真实正面概率是 \(\theta\)，则样本均值会随着 \(n\) 增大收敛到 \(\theta\)：

\[ \frac1n \sum_{i=1}^n X_i \xrightarrow{n\to\infty} \theta \]

📊 意义： ML 估计在这个例子里是合理的——样本越多，估计越准。

4. 为什么进入最小二乘

很多工程问题可以写成：

\[ y = H x \]

其中 \(y\) 是观测，\(H\) 是已知矩阵，\(x\) 是未知量。

但现实中有噪声，所以常常没有任何 \(x\) 能让方程完全成立。于是我们改问：

🎯 哪个 \(x\) 让 \(Hx\) 离 \(y\) 最近？——这就是最小二乘。

5. 线性方程是否有解

方程 \(y = Hx\) 有解当且仅当：

\[ y \in R(H) \]

其中 \(R(H)\) 是 \(H\) 的列空间——\(H\) 的所有列向量能线性组合出来的空间。

【补充例题】列空间

📐 有解

\(H = \begin{bmatrix}1\\2\end{bmatrix}\)，\(y = \begin{bmatrix}2\\4\end{bmatrix}=2H\)，有解。

❌ 无解

\(y = \begin{bmatrix}2\\5\end{bmatrix}\) 不在列空间直线上，没有精确解。

6. 最小二乘定义

若没有精确解，定义：

\[ \hat x = \arg\min_x \|y - Hx\|^2 \]

残差：

\[ r = y - H\hat x \]

最小二乘就是让残差长度最小。

7. 正规方程推导

定义代价函数：

\[ J(x) = \|y - Hx\|^2 = (y-Hx)^T (y-Hx) \]

展开：

\[ J(x) = y^T y - 2x^T H^T y + x^T H^T H x \]

对 \(x\) 求导并令为 0：

\[ \nabla J(x) = -2 H^T y + 2 H^T H x = 0 \]

得到 Normal Equation（正规方程）：

\[ \boxed{H^T H \hat x = H^T y} \]

8. 满列秩时的解

如果 \(H\) 满列秩，则 \(H^T H\) 可逆：

\[ \boxed{\hat x = (H^T H)^{-1} H^T y} \]

这就是最常见的最小二乘公式。

9. 不满秩时发生什么

如果 \(H\) 不满列秩，存在非零 \(x_c\) 使得 \(H x_c = 0\)。

若 \(\hat x\) 是一个最小二乘解，那么 \(\hat x + x_c\) 也是解，因为：

\[ H(\hat x + x_c) = H\hat x \]

⚠️ 重要： 不满秩时，解不唯一。需要额外的约束（如最小范数解）来选出特定的解。

10. 最小二乘的几何图像

\(H\hat x\) 是 \(y\) 在 \(R(H)\) 上的投影。

\[ H\hat x = \operatorname{Proj}_{R(H)}(y) \]

残差 \(y - H\hat x\) 垂直于 \(R(H)\)。

🔑 核心思想： 这句话是后面所有内容的核心——因为 LMMSE 和 Kalman Filter 也是投影，只不过投影空间变成了随机变量空间 📐

📌 小结

🪙 硬币 ML

\(\hat\theta_{\text{ML}} = \frac1n \sum x_i\)
样本均值 = 最大似然

📐 正规方程

\(H^T H \hat x = H^T y\)
满秩解：\(\hat x = (H^T H)^{-1} H^T y\)

🖼️ 几何

投影 + 正交
LMMSE 和 KF 都是这个思想的延伸

📝 自测题

  1. 偏硬币 ML 为什么等于样本均值？
  2. \(y \in R(H)\) 表示什么？
  3. 为什么正规方程说明残差和 \(H\) 的列空间正交？
查看答案提示

1. 对数似然求导令为 0 推导出样本均值。直观上 ML 选择使数据最可能的参数，对于伯努利分布就是样本中正面的比例。

2. 表示 \(y\) 可以表示为 \(H\) 各列的线性组合，即方程有精确解。

3. \(H^T(y - H\hat x) = 0\) 意味着残差与 \(H\) 的每一列都正交，因此与整个列空间正交。
