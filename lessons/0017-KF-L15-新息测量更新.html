1. 回顾：KF 的五个测量更新等式

上节课我们搭建了 Kalman Filter 的舞台——系统模型、基本假设与符号体系。本讲正式进入测量更新（Measurement Update）的详细推导。

当新的测量 \\(y_k\\) 到来时，我们需要把预测 \\(\\hat x_{k|k-1}\\) 更新为估计 \\(\\hat x_{k|k}\\)。更新过程由以下五个等式描述：

\\[ \\begin{aligned} \\text{①}\\quad & e_k = y_k - C \\hat x_{k|k-1} \\\\ \\text{②}\\quad & R_{e,k} = \\langle e_k, e_k \\rangle = C P_{k|k-1} C' + R \\\\ \\text{③}\\quad & \\hat x_{k|k} = \\hat x_{k|k-1} + K_k e_k \\\\ \\text{④}\\quad & K_k = P_{k|k-1} C' R_{e,k}^{-1} \\\\ \\text{⑤}\\quad & P_{k|k} = P_{k|k-1} - P_{k|k-1} C' R_{e,k}^{-1} C P_{k|k-1} \\end{aligned} \\]

本讲的核心任务就是逐个证明这五个等式。证明将以正交投影（Orthogonal Projection）为基本工具。

2. 预备知识：空间包含关系与噪声正交性

在开始证明之前，我们需要弄清楚各个随机变量所在的空间，以及它们之间的正交性关系。

状态的展开

系统的状态可以一直往前展开：

\\[ x_k = A^k x_0 + A^{k-1} w_0 + A^{k-2} w_1 + \\cdots + A w_{k-2} + w_{k-1} \\]

所以 \\(x_k\\) 是 \\(x_0, w_0, \\dots, w_{k-1}\\) 的线性函数。

空间包含关系

\\(\\mathcal{L}_k = \\text{span}\\{y_0, \\dots, y_k\\}\\)

测量空间 \\(\\mathcal{L}_k\\) 是由截至时刻 \\(k\\) 的所有测量生成的线性空间。

\\(\\mathcal{L}_k \\subset \\text{span}\\{x_0, w_0, \\dots, w_{k-1}; v_0, \\dots, v_k\\}\\)

\\(\\mathcal{L}_{k-1} = \\text{span}\\{y_0, \\dots, y_{k-1}\\}\\)

历史测量空间包含于更大的空间：

\\(\\mathcal{L}_{k-1} \\subset \\text{span}\\{x_0, w_0, \\dots, w_{k-2}; v_0, \\dots, v_{k-1}\\}\\)

关键正交性

💡 核心观察

\\(v_k\\) 与 \\(\\mathcal{L}_{k-1}\\) 正交！因为 \\(v_k\\) 与所有 \\(x_0, w_0, \\dots, w_{k-2}, v_0, \\dots, v_{k-1}\\) 都不相关，而 \\(\\mathcal{L}_{k-1}\\) 包含在这个大空间里。

这个观察将反复出现在下面的证明中。

3. ① 新息：\\(e_k = y_k - C\\hat x_{k|k-1}\\)

根据 Gram-Schmidt 正交化的思路，新息的定义是：

\\[ e_k = y_k - \\text{proj}\\{y_k \\mid \\mathcal{L}_{k-1}\\} \\]

即在当前测量中，去掉能被历史信息解释的部分后剩下的"纯新"部分。

证明

将 \\(y_k = C x_k + v_k\\) 代入：

\\[ \\begin{aligned} e_k &= y_k - \\sum_{j=0}^{k-1} \\langle y_k, e_j \\rangle \\|e_j\\|^{-2} e_j \\\\ &= y_k - \\sum_{j=0}^{k-1} \\langle C x_k + v_k, e_j \\rangle \\|e_j\\|^{-2} e_j \\\\ &= y_k - C \\sum_{j=0}^{k-1} \\langle x_k, e_j \\rangle \\|e_j\\|^{-2} e_j - \\sum_{j=0}^{k-1} \\langle v_k, e_j \\rangle \\|e_j\\|^{-2} e_j \\end{aligned} \\]

注意两个关键事实：

  • \\(\\text{proj}\\{x_k \\mid \\mathcal{L}_{k-1}\\} = \\hat x_{k|k-1}\\) —— 这是预测的定义
  • \\(\\langle v_k, e_j \\rangle = 0\\) —— 因为 \\(v_k \\perp \\mathcal{L}_{k-1}\\)，而 \\(e_j \\in \\mathcal{L}_{k-1}\\)

因此：

\\[ e_k = y_k - C \\hat x_{k|k-1} \\]

📌 直观理解

新息 = 真实测量 - 预测测量。它表示"新测量中到底是什么是我之前没想到的"。

4. ② 新息协方差：\\(R_{e,k} = C P_{k|k-1} C' + R\\)

新息协方差是新息 \\(e_k\\) 与自身的内积：

\\[ R_{e,k} = \\langle e_k, e_k \\rangle \\]

证明

由等式①：\\(e_k = y_k - C \\hat x_{k|k-1}\\)。代入 \\(y_k\\)：

\\[ \\begin{aligned} e_k &= C x_k + v_k - C \\hat x_{k|k-1} \\\\ &= C \\tilde x_{k|k-1} + v_k \\end{aligned} \\]

其中 \\(\\tilde x_{k|k-1} = x_k - \\hat x_{k|k-1}\\) 是预测误差。现在计算内积：

\\[ \\begin{aligned} R_{e,k} &= \\langle C \\tilde x_{k|k-1} + v_k,\\; C \\tilde x_{k|k-1} + v_k \\rangle \\\\ &= C \\underbrace{\\langle \\tilde x_{k|k-1}, \\tilde x_{k|k-1} \\rangle}_{P_{k|k-1}} C' + \\langle v_k, v_k \\rangle \\\\ &\\quad + \\underbrace{\\langle C \\tilde x_{k|k-1}, v_k \\rangle}_{=0} + \\underbrace{\\langle v_k, C \\tilde x_{k|k-1} \\rangle}_{=0} \\\\ &= C P_{k|k-1} C' + R \\end{aligned} \\]

交叉项为零的原因是：\\(\\tilde x_{k|k-1} \\in \\mathcal{L}_{k-1}\\)（因为它是预测误差，而预测在 \\(\\mathcal{L}_{k-1}\\) 中），而 \\(v_k \\perp \\mathcal{L}_{k-1}\\)。

💡 要点

新息协方差由两部分组成：预测不确定性通过观测矩阵映射到测量空间的部分 (\\(C P_{k|k-1} C'\\)) + 测量噪声本身的协方差 (\\(R\\))。

5. ③ - ④ 状态更新与 Kalman 增益

这两个等式一起给出测量更新后的状态估计：

\\[ \\hat x_{k|k} = \\hat x_{k|k-1} + K_k e_k, \\quad K_k = P_{k|k-1} C' R_{e,k}^{-1} \\]

步骤 1：计算 \\(\\langle x_k, e_k \\rangle\\)

为了将 \\(x_k\\) 投影到 \\(\\mathcal{L}_k\\)（包含了新息 \\(e_k\\)），我们需要知道 \\(x_k\\) 与 \\(e_k\\) 的内积。

\\[ \\begin{aligned} \\langle x_k, e_k \\rangle &= \\langle x_k,\\; y_k - C \\hat x_{k|k-1} \\rangle \\\\ &= \\langle x_k,\\; C \\tilde x_{k|k-1} + v_k \\rangle \\\\ &= \\langle x_k, C \\tilde x_{k|k-1} \\rangle + \\cancel{\\langle x_k, v_k \\rangle} \\\\ &= \\langle \\hat x_{k|k-1} + \\tilde x_{k|k-1},\\; C \\tilde x_{k|k-1} \\rangle \\\\ &= \\cancel{\\langle \\hat x_{k|k-1}, C \\tilde x_{k|k-1} \\rangle} + \\langle \\tilde x_{k|k-1}, C \\tilde x_{k|k-1} \\rangle \\\\ &= P_{k|k-1} C' \\end{aligned} \\]

其中交叉项 \\(\\langle \\hat x_{k|k-1}, \\tilde x_{k|k-1} \\rangle = 0\\) 正是正交原理（Orthogonality Principle）——最优估计的误差与估计本身正交。

步骤 2：投影到 \\(\\mathcal{L}_k\\)

现在将 \\(x_k\\) 投影到 \\(\\mathcal{L}_k\\)（它由 \\(\\mathcal{L}_{k-1}\\) 和 \\(e_k\\) 张成）：

\\[ \\begin{aligned} \\hat x_{k|k} &= \\text{proj}\\{x_k \\mid \\mathcal{L}_k\\} \\\\ &= \\sum_{j=0}^{k} \\langle x_k, e_j \\rangle \\|e_j\\|^{-2} e_j \\\\ &= \\underbrace{\\sum_{j=0}^{k-1} \\langle x_k, e_j \\rangle \\|e_j\\|^{-2} e_j}_{\\hat x_{k|k-1}} + \\langle x_k, e_k \\rangle \\|e_k\\|^{-2} e_k \\\\ &= \\hat x_{k|k-1} + P_{k|k-1} C' \\cdot R_{e,k}^{-1} \\cdot e_k \\end{aligned} \\]

定义 Kalman 增益 \\(K_k = P_{k|k-1} C' R_{e,k}^{-1}\\)，得到：

\\[ \\hat x_{k|k} = \\hat x_{k|k-1} + K_k e_k \\]

🧠 几何解释

Kalman 增益 \\(K_k\\) 控制着我们对新息的"信任程度"。如果预测很准（\\(P_{k|k-1}\\) 小），增益就小；如果测量噪声小（\\(R\\) 小），增益就大。

6. ⑤ 后验协方差更新

最后，我们推导更新后的误差协方差 \\(P_{k|k}\\)：

\\[ P_{k|k} = P_{k|k-1} - P_{k|k-1} C' R_{e,k}^{-1} C P_{k|k-1} \\]

证明

首先写出估计误差：

\\[ \\tilde x_{k|k} = x_k - \\hat x_{k|k} = \\tilde x_{k|k-1} - K_k e_k \\]

然后计算内积：

\\[ \\begin{aligned} P_{k|k} &= \\langle \\tilde x_{k|k},\\; \\tilde x_{k|k} \\rangle \\\\ &= \\langle \\tilde x_{k|k-1} - K_k e_k,\\; \\tilde x_{k|k-1} - K_k e_k \\rangle \\\\ &= P_{k|k-1} - K_k \\langle e_k, \\tilde x_{k|k-1} \\rangle - \\langle \\tilde x_{k|k-1}, e_k \\rangle K_k' + K_k \\langle e_k, e_k \\rangle K_k' \\end{aligned} \\]

我们需要计算其中的交叉项：

\\[ \\begin{aligned} \\langle \\tilde x_{k|k-1}, e_k \\rangle &= \\langle \\tilde x_{k|k-1},\\; C \\tilde x_{k|k-1} + v_k \\rangle = P_{k|k-1} C' \\\\ \\langle e_k, \\tilde x_{k|k-1} \\rangle &= C P_{k|k-1} \\end{aligned} \\]

代入并利用 \\(K_k = P_{k|k-1} C' R_{e,k}^{-1}\\)：

\\[ \\begin{aligned} P_{k|k} &= P_{k|k-1} - K_k \\cdot C P_{k|k-1} - P_{k|k-1} C' \\cdot K_k' + K_k \\cdot R_{e,k} \\cdot K_k' \\\\ &= P_{k|k-1} - P_{k|k-1} C' R_{e,k}^{-1} C P_{k|k-1} - P_{k|k-1} C' R_{e,k}^{-1} C P_{k|k-1} \\\\ &\\quad + P_{k|k-1} C' R_{e,k}^{-1} \\cdot R_{e,k} \\cdot R_{e,k}^{-1} C P_{k|k-1} \\\\ &= P_{k|k-1} - P_{k|k-1} C' R_{e,k}^{-1} C P_{k|k-1} \\end{aligned} \\]

或者用更紧凑的形式写出：

\\[ P_{k|k} = P_{k|k-1} - P_{k|k-1} C' [C P_{k|k-1} C' + R]^{-1} C P_{k|k-1} \\]

📌 关键认识

后验协方差 = 先验协方差 - 新信息带来的"不确定性降低量"。这个降低量正是 \\(K_k R_{e,k} K_k'\\)，也就是 Kalman 增益加权后的新息协方差。

7. 完整测量更新回顾

到此我们完成了五个等式的证明。完整的 KF 测量更新流程如下：

① 新息

\\[ e_k = y_k - C \\hat x_{k|k-1} \\]

② 新息协方差

\\[ R_{e,k} = C P_{k|k-1} C' + R \\]

③ Kalman 增益

\\[ K_k = P_{k|k-1} C' R_{e,k}^{-1} \\]

④ 状态更新

\\[ \\hat x_{k|k} = \\hat x_{k|k-1} + K_k e_k \\]

⑤ 协方差更新

\\[ P_{k|k} = (I - K_k C) P_{k|k-1} \\]

在证明过程中，我们反复用到了以下技巧：

  1. 分解技巧：把状态拆成预测 + 预测误差，利用正交原理消掉交叉项
  2. 空间论证：利用 \\(v_k \\perp \\mathcal{L}_{k-1}\\) 消去含 \\(v_k\\) 的交叉项
  3. 投影展开：把 \\(\\mathcal{L}_k\\) 上的投影拆成 \\(\\mathcal{L}_{k-1}\\) 上的投影 + 新息方向上的投影

🔭 下讲预告

测量更新完备之后，我们将进入 Time Update（时间更新）的推导，然后讨论 \\(P_{k|k}\\) 的收敛性、固定增益估计器以及传感器融合。

问题 1

点击查看答案

\\(e_k \\perp \\mathcal{L}_{k-1}\\)。因为 \\(e_k\\) 定义为 \\(y_k\\) 在 \\(\\mathcal{L}_{k-1}\\) 上投影的残差，根据正交原理，残差与投影空间正交。

问题 2

证明 \\(\\langle \\tilde x_{k|k-1}, v_k \\rangle = 0\\)。

点击查看答案

\\(\\tilde x_{k|k-1} \\in \\mathcal{L}_{k-1}\\)（预测误差在历史测量空间内），而 \\(v_k \\perp \\mathcal{L}_{k-1}\\)（\\(v_k\\) 与所有历史信息不相关）。所以 \\(\\tilde x_{k|k-1}\\) 与 \\(v_k\\) 正交。
