线性高斯下 MMSE 等价与 Kalman 增益优化

条件高斯分布推导 MMSE 估计 = LMMSE + 控制输入的融入 + 从优化角度重新推导 Kalman 增益

📅 2026-05-30 🧩 MMSE 📐 条件高斯 ⚡ 优化方法

🎯 本讲目标

三种方法推导 KF：(1) 概率法——用条件高斯公式证明 MMSE = LMMSE；(2) 控制输入融入——将 B u_k 加入 KF 框架；(3) 优化法——将 P_k 写成 K_k 的二次型并通过 completion of squares 求解。

🔑 核心思想

在线性高斯系统中，条件均值（MMSE）恰好等于投影（LMMSE）。控制输入不影响协方差传播，只影响均值。优化法则从另一个视角验证了 Kalman 增益的最优性。

1. 回顾：几何方法的局限

前两节课我们用几何方法（投影定理）推导了 KF 的五个核心等式，得到了 LMMSE（线性最小均方误差估计）。但有一个关键问题尚未回答：

❓ 核心问题

在线性高斯系统中，LMMSE 是否等于 MMSE（最小均方误差估计，即条件均值 \\(E[x_k | y_0, \\dots, y_k]\\)）？

如果是，那么就说明我们在 KF 框架中使用的线性估计实际上是最优的——没有任何非线性方法能做得更好。

本讲首先用概率方法（条件高斯分布）来回答这个问题，然后再从优化角度重新验证 Kalman 增益的最优性。

2. 系统模型与符号体系（含控制输入）

现在我们将控制输入 \\(u_k\\) 加入到系统模型中：

状态方程

\\[ x_{k+1} = A x_k + B u_k + w_k \\]

\\(w_k \\sim N(0, Q)\\)

测量方程

\\[ y_k = C x_k + v_k \\]

\\(v_k \\sim N(0, R)\\)

符号体系

定义信息集合：

\\[ Y_k = \\{y_0, y_1, \\dots, y_k\\}, \\quad U_k = \\{u_0, u_1, \\dots, u_k\\} \\]

MMSE 估计定义为条件均值：

估计

\\[ \\hat x_{k|k} = E[x_k \\mid Y_k, U_{k-1}] \\] \\[ P_{k|k} = \\langle x_k - \\hat x_{k|k},\\; x_k - \\hat x_{k|k} \\rangle \\]

预测

\\[ \\hat x_{k|k-1} = E[x_k \\mid Y_{k-1}, U_{k-1}] \\] \\[ P_{k|k-1} = \\langle x_k - \\hat x_{k|k-1},\\; x_k - \\hat x_{k|k-1} \\rangle \\]

💡 关于 U_{k-1} 的说明

为什么是 \\(U_{k-1}\\) 而不是 \\(U_k\\)？因为 \\(u_k\\) 是在估计 \\(x_k\\) 之后才施加的控制，它影响的是 \\(x_{k+1}\\)，而不是 \\(x_k\\)。所以估计 \\(x_k\\) 时只能用过去的控制输入。

3. 概率方法：条件高斯定理

首先回顾概率论中的一个核心定理：

定理：设 \\(X\\) 和 \\(Y\\) 联合高斯：

\\[ \\begin{bmatrix} X \\\\ Y \\end{bmatrix} \\sim N\\left( \\begin{bmatrix} \\bar x \\\\ \\bar y \\end{bmatrix},\\; \\begin{bmatrix} \\Sigma_x & \\Sigma_{xy} \\\\ \\Sigma_{yx} & \\Sigma_y \\end{bmatrix} \\right) \\]

则在给定 \\(Y = y\\) 条件下，\\(X\\) 仍然是高斯分布：

\\[ \\boxed{\\mu_{x|y=y} = \\bar x + \\Sigma_{xy} \\Sigma_y^{-1} (y - \\bar y)} \\] \\[ \\boxed{\\Sigma_{x|y=y} = \\Sigma_x - \\Sigma_{xy} \\Sigma_y^{-1} \\Sigma_{yx}} \\]

特例验证

① \\(X = Y\\) 时

\\[ \\mu_{x|y=y} = y,\\quad \\Sigma_{x|y=y} = 0 \\]

完全确定，不确定性降为零。

② \\(X \\perp Y\\) 时

\\[ \\mu_{x|y=y} = \\bar x,\\quad \\Sigma_{x|y=y} = \\Sigma_x \\]

Y 不提供 X 的任何信息，条件不变。

🎯 重要意义

这个定理告诉我们：在线性高斯系统中，给定测量后的条件均值是测量的线性函数。而 LMMSE 恰好就是那个线性函数——所以 MMSE = LMMSE！

4. Time Update（概率法）

给定 \\(\\hat x_{k-1|k-1}\\) 和 \\(P_{k-1|k-1}\\)，计算预测：

预测均值

\\[ \\begin{aligned} \\hat x_{k|k-1} &= E[x_k \\mid Y_{k-1}, U_{k-1}] \\\\ &= E[A x_{k-1} + B u_{k-1} + w_{k-1} \\mid Y_{k-1}, U_{k-1}] \\\\ &= A E[x_{k-1} \\mid Y_{k-1}, U_{k-1}] + B u_{k-1} + E[w_{k-1} \\mid Y_{k-1}, U_{k-1}] \\\\ &= A \\hat x_{k-1|k-1} + B u_{k-1} \\end{aligned} \\]

其中 \\(E[w_{k-1} \\mid Y_{k-1}, U_{k-1}] = 0\\) 因为 \\(w_{k-1}\\) 与历史信息不相关。

预测协方差

\\[ \\begin{aligned} P_{k|k-1} &= E\\big[(x_k - \\hat x_{k|k-1})(x_k - \\hat x_{k|k-1})' \\mid Y_{k-1}, U_{k-1}\\big] \\\\ &= E\\big[(A \\tilde x_{k-1|k-1} + w_{k-1})(A \\tilde x_{k-1|k-1} + w_{k-1})' \\mid Y_{k-1}, U_{k-1}\\big] \\\\ &= A P_{k-1|k-1} A' + Q \\end{aligned} \\]

📌 关键观察

控制输入 \\(B u_k\\) 只影响预测均值，不影响预测协方差。这意味着控制与估计可以分离（Separation Principle）——设计控制器时可以暂时忽略估计误差。

5. Measurement Update（概率法）

给定 \\(\\hat x_{k|k-1}\\) 和 \\(P_{k|k-1}\\)，新的测量 \\(y_k\\) 到来。我们需要找到 \\(E[x_k \\mid Y_k, U_{k-1}]\\)。

步骤 1：计算条件期望

\\[ E[y_k \\mid Y_{k-1}, U_{k-1}] = C \\hat x_{k|k-1} \\]

步骤 2：计算条件协方差

\\[ \\begin{aligned} &E\\big[(y_k - E[y_k \\mid Y_{k-1}, U_{k-1}])(\\cdot)' \\big] \\\\ &= E\\big[(C \\tilde x_{k|k-1} + v_k)(C \\tilde x_{k|k-1} + v_k)'\\big] \\\\ &= C P_{k|k-1} C' + R \\end{aligned} \\]

步骤 3：计算交叉协方差

\\[ E\\big[(x_k - \\hat x_{k|k-1})(y_k - C \\hat x_{k|k-1})' \\mid Y_{k-1}, U_{k-1}\\big] = P_{k|k-1} C' \\]

步骤 4：拼装联合高斯分布

\\[ \\begin{bmatrix} x_k \\\\ y_k \\end{bmatrix} \\Bigg| Y_{k-1}, U_{k-1} \\sim N\\left( \\begin{bmatrix} \\hat x_{k|k-1} \\\\ C \\hat x_{k|k-1} \\end{bmatrix},\\; \\begin{bmatrix} P_{k|k-1} & P_{k|k-1} C' \\\\ C P_{k|k-1} & C P_{k|k-1} C' + R \\end{bmatrix} \\right) \\]

步骤 5：套用条件高斯公式

\\[ \\boxed{\\hat x_{k|k} = \\hat x_{k|k-1} + P_{k|k-1} C' [C P_{k|k-1} C' + R]^{-1} (y_k - C \\hat x_{k|k-1})} \\] \\[ \\boxed{P_{k|k} = P_{k|k-1} - P_{k|k-1} C' [C P_{k|k-1} C' + R]^{-1} C P_{k|k-1}} \\]

与几何方法得到的表达式完全一致！

6. MMSE = LMMSE：线性高斯下的等价性

现在我们可以回答核心问题了：

✅ 结论

在线性高斯系统中，MMSE 估计等于 LMMSE 估计。

为什么？

因为：

  1. MMSE 定义：\\(\\hat x_{k|k}^{\\text{MMSE}} = E[x_k \\mid Y_k, U_{k-1}]\\)，即条件均值
  2. 条件高斯公式告诉我们，在线性高斯系统中，条件均值是测量的线性函数
  3. LMMSE 定义：在所有的线性估计器中找均方误差最小的那个
  4. 既然 MMSE 本身（条件均值）已经是线性函数，那么它恰好就是 LMMSE 的最优解

📐 几何方法

投影 → LMMSE

只利用了内积空间结构

📊 概率方法

条件均值 → MMSE

利用了高斯分布的性质

🎯 所以

在线性高斯情况下，\\(\\hat x_{k|k}^{\\text{LMMSE}} = \\hat x_{k|k}^{\\text{MMSE}}\\)。两者完美重合。这在非高斯系统中一般不成立——非高斯条件下 MMSE 通常小于 LMMSE（因为 MMSE 在所有估计器中最优，而 LMMSE 只在线性估计器中最优）。

7. 优化方法：从 \\(P_k(K_k)\\) 到最优增益

第三种推导 KF 的方法是从优化的角度——把 \\(P_k\\) 写成 \\(K_k\\) 的函数，然后最小化。

设定

考虑如下递归线性估计器：

\\[ \\hat x_k = A \\hat x_{k-1} + K_k (y_k - C A \\hat x_{k-1}) \\]

初始条件 \\(\\hat x_0 = 0\\)。这是一个"预测 + 修正"结构的线性估计器，但增益 \\(K_k\\) 尚未确定。

优化问题

\\[ \\boxed{\\text{Find } K_k \\text{ s.t. } P_k = E[(x_k - \\hat x_k)(x_k - \\hat x_k)'] \\text{ is minimized}} \\]

推导思路（Take-home Exercise）

基本步骤：

  1. 将 \\(x_k\\) 和 \\(\\hat x_k\\) 都用系统方程展开
  2. 得到 \\(x_k - \\hat x_k\\) 的表达式（含 \\(K_k\\)）
  3. 计算 \\(P_k = E[(x_k - \\hat x_k)(x_k - \\hat x_k)']\\)
  4. \\(P_k\\) 会成为 \\(K_k\\) 的二次型
  5. 用 Completion of Squares 方法求出最优 \\(K_k\\)
\\[ \\boxed{K_k = P_{k|k-1} C' [C P_{k|k-1} C' + R]^{-1}} \\]

得到的解与几何法、概率法完全一致！

⚡ 优化法的独特价值

优化法不依赖几何直觉（正交投影）也不依赖概率假设（高斯分布），只需要"构造一个形式合理的估计器 + 最小化均方误差"这一工程直觉。这在推广到非线性/非高斯系统时特别有价值（例如你可以改变估计器的结构来适应非线性）。

8. 三种方法总结

到此我们学习了三种推导 KF 的方法：

方法 核心工具 优势 局限
📐 几何法 投影定理、正交原理 直观、可视化 只得到 LMMSE
📊 概率法 条件高斯公式 证明 MMSE = LMMSE 需要高斯假设
⚡ 优化法 二次型最小化 工程直觉强、可推广 需预设估计器结构

三种方法都得到了完全相同的五个等式，这说明：

🌟 Kalman Filter 的普适性

无论你从哪个角度进入——几何、概率还是优化——KF 都是在线性高斯系统下你能做到的最好估计器。这种"殊途同归"的美，正是 KF 作为经典算法的魅力所在。

📝 自测题

问题 1

为什么在线性高斯系统中 MMSE = LMMSE？

点击查看答案

在线性高斯系统中，给定测量后的条件均值（MMSE）是测量的线性函数（由条件高斯公式保证）。而 LMMSE 是所有线性估计器中 MSE 最小的。既然 MMSE 本身也是线性的，它必然就是那个最优的线性估计器。

问题 2

控制输入 \\(B u_k\\) 对 KF 的协方差 \\(P_{k|k}\\) 有影响吗？为什么？

点击查看答案

没有影响。控制输入 \\(B u_k\\) 在给定 \\(U_{k-1}\\) 条件下是确定量，它只影响预测均值，不影响误差和协方差。这是 Separation Principle（分离原理）的基础——控制律设计和估计器设计可以独立进行。

问题 3

在非高斯系统中，MMSE 和 LMMSE 哪个更小？
