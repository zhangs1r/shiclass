1. 回顾：KF 完整五等式

前两讲我们从几何投影角度完成了测量更新的五个等式证明。现在回顾完整的 KF 递归方程组：

⏱ Time Update（预测步）

\$$ $\\hat x_{k+1|k} = A \\hat x_{k|k}$ \$$ \$$ P_{k+1|k} = A P_{k|k} A' + Q \$$

📏 Measurement Update（更新步）

\$$ $e_{k} = y_{k} - C \\hat x_{k|k-1}$ \$$ \$$ K_{k} = P_{k|k-1} C' [C P_{k|k-1} C' + R]^{-1} \$$ \$$ $\\hat x_{k|k} = \\hat x_{k|k-1} + K_{k} e_{k}$ \$$ \$$ P_{k|k} = P_{k|k-1} - K_{k} C P_{k|k-1} \$$

本讲的任务是证明 Time Update 的两个等式（⑥ - ⑦），并分析其重要性质。

2. 时间更新推导（⑥ - ⑦）

定理：给定 \$\\hat x_{k|k}\$ 和 \$P_{k|k}\$，时间更新为：

\$$ $\\boxed{\\hat x_{k+1|k} = A \\hat x_{k|k}}$ \\qquad \\boxed{P_{k+1|k} = A P_{k|k} A' + Q} \$$

证明：预测均值 \$\\hat x_{k+1|k}\$

预测均值是 \$x_{k+1}\$ 在当前测量空间 \$\mathcal{L}_k\$ 上的投影（条件期望）：

\$$ \\begin{aligned} $\\hat x_{k+1|k} &= \\text{proj}\\{x_{k+1} \\mid \\mathcal{L}_k\\} \\\\\\\\$ $&= \\text{proj}\\{A x_{k} + w_{k} \\mid \\mathcal{L}_k\\} \\\\\\\\$ $&= \\text{proj}\\{A x_{k} \\mid \\mathcal{L}_k\\} + \\text{proj}\\{w_{k} \\mid \\mathcal{L}_k\\} \\\\\\\\$ $&= A \\cdot \\text{proj}\\{x_{k} \\mid \\mathcal{L}_k\\} + 0 \\\\\\\\$ $&= A \\hat x_{k|k}$ \\end{aligned} \$$

其中 \$\\text{proj}\\{w_{k} \\mid \\mathcal{L}_k\\} = 0\$ 因为 \$w_{k}\$ 与所有历史测量不相关——\$w_{k}\$ 只影响未来的状态，不影响当前及过去的测量。

证明：预测误差协方差 \$P_{k+1|k}\$

首先计算预测误差：

\$$ \\begin{aligned} $\\tilde x_{k+1|k} &= x_{k+1} - \\hat x_{k+1|k} \\\\\\\\$ $&= (A x_{k} + w_{k}) - A \\hat x_{k|k} \\\\\\\\$ $&= A \\tilde x_{k|k} + w_{k}$ \\end{aligned} \$$

然后计算内积：

\$$ \\begin{aligned} $P_{k+1|k} &= \\langle \\tilde x_{k+1|k},\\; \\tilde x_{k+1|k} \\rangle \\\\\\\\$ $&= \\langle A \\tilde x_{k|k} + w_{k},\\; A \\tilde x_{k|k} + w_{k} \\rangle \\\\\\\\$ $&= A \\underbrace{\\langle \\tilde x_{k|k}, \\tilde x_{k|k} \\rangle}_{P_{k|k}} A' + \\underbrace{\\langle w_{k}, w_{k} \\rangle}_{Q} \\\\\\\\$ $&\\quad + \\underbrace{A \\langle \\tilde x_{k|k}, w_{k} \\rangle}_{=0} + \\underbrace{\\langle w_{k}, \\tilde x_{k|k} \\rangle A'}_{=0} \\\\\\\\$ &= A P_{k|k} A' + Q \\end{aligned} \$$

交叉项为零是因为 \$w_{k} \\perp \\mathcal{L}_k\$，而 \$\\tilde x_{k|k} \\in \\mathcal{L}_k\$。

💡 直观理解

时间更新就是在没有新测量的情况下，用系统模型"推进一步"：
① 均值通过 \$A\$ 线性传播
② 不确定性通过 \$A\$ 传播后，还要加上过程噪声 \$Q\$——因为有新的随机扰动进入了系统。

3. 从几何角度理解时间更新

时间更新的几何意义很清晰：

📐 均值传播

\$\\hat x_{k+1|k} = A \\hat x_{k|k}\$

状态空间中的线性映射：已知 \$x_{k}\$ 在 \$\mathcal{L}_k\$ 上的投影，通过 \$A\$ 变换得到 \$x_{k+1}\$ 在同一空间上的投影。

📐 不确定性传播

\$P_{k+1|k} = A P_{k|k} A' + Q\$

椭球通过 \$A\$ 进行线性变换后，再叠加一个 \$Q\$ 椭球（因为 \$w_{k}\$ 的加入）。

🔭 注意

时间更新后，不确定性是增大的：\$P_{k+1|k} \\ge P_{k|k}\$（在矩阵半正定意义下）。因为在没有新测量的时间段里，过程噪声不断累积，我们的认知在衰减。等到下一次测量到来时，测量更新会重新降低不确定性。

4. 预测与估计的完整递归步

现在我们可以把完整的 KF 递归写成紧凑形式。给定 \$\\hat x_{k-1|k-1}\$ 和 \$P_{k-1|k-1}\$：

预测步（时间更新）

\$$ \\begin{aligned} $\\hat x_{k|k-1} &= A \\hat x_{k-1|k-1} \\\\\\\\$ P_{k|k-1} &= A P_{k-1|k-1} A' + Q \\end{aligned} \$$

更新步（测量更新）

\$$ \\begin{aligned} K_{k} &= P_{k|k-1} C' [C P_{k|k-1} C' + R]^{-1} \\\\\\\\ $\\hat x_{k|k} &= \\hat x_{k|k-1} + K_{k} (y_{k} - C \\hat x_{k|k-1}) \\\\\\\\$ P_{k|k} &= P_{k|k-1} - K_{k} C P_{k|k-1} \\end{aligned} \$$

🔄 递归节奏

预测 → 测量到来 → 更新 → 再预测 → 这是一个永不停息的循环。每来一个新的测量，只需要 O(n³) 的矩阵运算（n 为状态维度）。对于许多实际系统，这可以做到实时运行。

5. 协方差的可离线计算特性

观察 KF 的递归等式，可以发现一个重要性质：

\$$ \\begin{aligned} $P_{k|k-1} &\\;\\text{only depends on}\\; P_{k-1|k-1} \\\\\\\\$ $K_{k} &\\;\\text{only depends on}\\; P_{k|k-1} \\\\\\\\$ $P_{k|k} &\\;\\text{only depends on}\\; P_{k|k-1} \\\\\\\\$ \\end{aligned} \$$

特别地，\$P_{k|k}\$ 和 \$K_{k}\$ 完全不依赖测量值 \$y_{k}\$！它们只由系统参数 \$(A, C, Q, R)\$ 和初始协方差 \$\\Pi_{0}\$ 决定。

🚀 离线阶段

预先计算所有 \$P_{k|k}\$ 和 \$K_{k}\$，存成表格。只需要矩阵运算，不涉及传感器数据。

⚡ 在线阶段

查表获得 \$K_{k}\$，然后只做简单的向量运算更新状态估计。计算量从 O(n³) 降到 O(n²)。

这在嵌入式系统、卫星导航等计算资源有限的场景中是巨大的工程优势。

6. \$P_{k|k}\$ 的收敛性分析

📊 模拟实验显示：\$P_{k|k}\$ 随 \$k\$ 增加而趋于一个稳态值 \$\\bar P\$

收敛条件

随着时间 \$k \\to \\infty\$，\$P_{k|k}\$ 是否会收敛到一个稳态值？答案是肯定的，但需要满足以下两个条件：

📌 可观性（Observability）

\$$ $(A, C)\\; \\text{observable}$ \$$

系统的所有模态都能通过测量被观测到。如果某个模态不可观，它的不确定性永远不会降低。

📌 能控性（Controllability）

\$$ $(A, \\sqrt{Q})\\; \\text{controllable}$ \$$

过程噪声能激励系统的所有模态。如果某个模态不受噪声驱动，它的不确定性就是"冻结"的。

收敛的工程意义

当 \$P_{k|k}\$ 收敛到 \$\\bar P\$ 之后，Kalman 增益也收敛到一个常数矩阵 \$K^*\$。此时估计器退化为一个简单的线性时不变系统：

\$$ $\\hat x_{k+1|k} = A \\hat x_{k|k-1} + K^* [y_{k} - C \\hat x_{k|k-1}]$ \$$

🎯 重要含义

收敛意味着我们不仅不需要在线计算增益，甚至不需要查表——只需要记住一个固定的增益矩阵 \$K^*\$。这大幅降低了实现复杂度，是 KF 工程化部署的关键。

7. 可观性与能控性的直观理解

这两个概念是控制理论的基础，让我们直观理解：

🔍 可观性

你能否通过测量 \$y_{k}\$ 推断出状态 \$x_{k}\$ 的每一个分量？

例子：如果 \$C\$ 的第 \$i\$ 列全为零，那么 \$x_{k}\$ 的第 \$i\$ 个分量永远无法被观测到——即使有再多的测量也无法降低其不确定性。

🎛️ 能控性

过程噪声 \$w_{k}\$ 能否激励状态的每一个分量？

例子：如果 \$Q\$ 的某个对角元为零，对应的状态分量不受噪声驱动——它的不确定性变化仅由 \$A\$ 传播决定。

💡 对比记忆

可观性 + 能控性 = 系统每个模态既"看得见"又"被激励"。满足这两个条件时，KF 的不确定性最终会趋于一个稳态值，估计误差的方差不会再变化。

📝 自测题

问题 1

为什么 KF 的协方差 \$P_{k|k}\$ 和增益 \$K_{k}\$ 与测量值 \$y_{k}\$ 无关？

点击查看答案

因为 \$P_{k|k}\$ 的更新只依赖于系统参数 \$(A, C, Q, R)\$ 和上一步的 \$P_{k-1|k-1}\$，不包含任何测量值。测量值只影响状态估计 \$\\hat x_{k|k}\$ 的轨迹，不影响不确定性。

问题 2

为什么时间更新后协方差 \$P_{k+1|k}\$ 会比 \$P_{k|k}\$ 更大？

点击查看答案

因为时间更新引入了过程噪声 \$w_{k}\$ 带来的不确定性 \$Q\$。在没有新测量来修正之前，系统的不确定性只会增加不会减少。等到测量更新时，新信息才会降低不确定性。
