1. 回顾：几何方法的局限

前几节课我们用几何方法（投影定理）推导了 KF 的五个核心等式，得到了 LMMSE（线性最小均方误差估计）。但有一个关键问题尚未回答：

❓ 核心问题

在线性高斯系统中，LMMSE 是否等于 MMSE（最小均方误差估计，即条件均值 \$E[x_k | y_0, \\dots, y_k]\$）？

回答这个问题至关重要：

  • 如果 是：那么 KF 不仅是所有线性估计器中最好的，更是所有估计器（包括非线性）中最好的
  • 如果 否：那么我们可能需要寻找非线性方法来做更好的估计

本讲将用概率方法（条件高斯分布）来回答这个问题。顺便，我们也会在系统模型中引入控制输入 \$B u_k\$。

2. 系统模型与符号体系（含控制输入）

现在我们将控制输入 \$u_k\$ 加入到系统模型中：

状态方程

\$$ x_{k+1} = A x_k + B u_k + w_k \$$

\$w_k \\sim N(0, Q)\$，\$x_0 \\sim N(\\bar x_0, \\Pi_0)\$

测量方程

\$$ y_k = C x_k + v_k \$$

\$v_k \\sim N(0, R)\$，与 \$w_k\$、\$x_0\$ 独立

符号体系

定义信息集合：

\$$ Y_k = \\{y_0, y_1, \\dots, y_k\\}, \\quad U_k = \\{u_0, u_1, \\dots, u_k\\} \$$

MMSE 估计定义为条件均值：

估计

\$$ \\hat x_{k|k} = E[x_k \\mid Y_k, U_{k-1}] \$$ \$$ P_{k|k} = \\text{Cov}[x_k \\mid Y_k, U_{k-1}] \$$

预测

\$$ \\hat x_{k|k-1} = E[x_k \\mid Y_{k-1}, U_{k-1}] \$$ \$$ P_{k|k-1} = \\text{Cov}[x_k \\mid Y_{k-1}, U_{k-1}] \$$

💡 关于 \$U_{k-1}\$ 的说明

为什么是 \$U_{k-1}\$ 而不是 \$U_k\$？因为 \$u_k\$ 是在估计 \$x_k\$ 之后才施加的控制，它影响的是 \$x_{k+1}\$，而不是 \$x_k\$。所以估计 \$x_k\$ 时只能用过去的控制输入。

3. 条件高斯定理回顾

这是整个概率方法推导 KF 的核心定理：

定理：设 \$X\$ 和 \$Y\$ 联合高斯：

\$$ \\begin{bmatrix} X \\\\ Y \\end{bmatrix} \\sim N\\left( \\begin{bmatrix} \\bar x \\\\ \\bar y \\end{bmatrix},\\; \\begin{bmatrix} \\Sigma_x & \\Sigma_{xy} \\\\ \\Sigma_{yx} & \\Sigma_y \\end{bmatrix} \\right) \$$

则在给定 \$Y = y\$ 条件下，\$X\$ 仍然是高斯分布：

\$$ \\boxed{\\mu_{x|y=y} = \\bar x + \\Sigma_{xy} \\Sigma_y^{-1} (y - \\bar y)} \$$ \$$ \\boxed{\\Sigma_{x|y=y} = \\Sigma_x - \\Sigma_{xy} \\Sigma_y^{-1} \\Sigma_{yx}} \$$

特例验证

① \$X = Y\$ 时

\$$ \\mu_{x|y=y} = y,\\quad \\Sigma_{x|y=y} = 0 \$$

完全确定，不确定性降为零。

② \$X \\perp Y\$ 时

\$$ \\mu_{x|y=y} = \\bar x,\\quad \\Sigma_{x|y=y} = \\Sigma_x \$$

Y 不提供 X 的任何信息。

🎯 重要意义

注意条件均值 \$\\bar x + \\Sigma_{xy} \\Sigma_y^{-1} (y - \\bar y)\$ 是 \$y\$ 的线性函数。而 LMMSE 是在所有线性估计器中最小化 MSE 的解——它就是那个线性函数！所以在线性高斯系统中，MMSE = LMMSE。

4. Time Update（概率法）

给定 \$\\hat x_{k-1|k-1}\$ 和 \$P_{k-1|k-1}\$，计算预测：

预测均值

\$$ \\begin{aligned} \\hat x_{k|k-1} &= E[x_k \\mid Y_{k-1}, U_{k-1}] \\\\\\\\ &= E[A x_{k-1} + B u_{k-1} + w_{k-1} \\mid Y_{k-1}, U_{k-1}] \\\\\\\\ &= A E[x_{k-1} \\mid Y_{k-1}, U_{k-1}] + B u_{k-1} + E[w_{k-1} \\mid Y_{k-1}, U_{k-1}] \\\\\\\\ &= A \\hat x_{k-1|k-1} + B u_{k-1} \\end{aligned} \$$

其中 \$E[w_{k-1} \\mid Y_{k-1}, U_{k-1}] = 0\$ 因为 \$w_{k-1}\$ 与历史信息不相关（零均值高斯，且独立于所有历史信息）。

预测协方差

\$$ \\begin{aligned} P_{k|k-1} &= E\\big[(x_k - \\hat x_{k|k-1})(x_k - \\hat x_{k|k-1})' \\mid Y_{k-1}, U_{k-1}\\big] \\\\\\\\ &= E\\big[(A \\tilde x_{k-1|k-1} + w_{k-1})(A \\tilde x_{k-1|k-1} + w_{k-1})' \\mid Y_{k-1}, U_{k-1}\\big] \\\\\\\\ &= A P_{k-1|k-1} A' + Q \\end{aligned} \$$

交叉项 \$E[\\tilde x_{k-1|k-1} w_{k-1}'] = 0\$ 因为 \$w_{k-1}\$ 与历史信息独立。

📌 关键观察

控制输入 \$B u_k\$ 只影响预测均值，不影响预测协方差。这意味着控制与估计可以分离（Separation Principle）——设计控制器时可以暂时忽略估计误差。

5. Measurement Update（概率法）

给定 \$\\hat x_{k|k-1}\$ 和 \$P_{k|k-1}\$，新的测量 \$y_k\$ 到来。我们需要找到 \$E[x_k \\mid Y_k, U_{k-1}]\$。

步骤 1：构造联合高斯分布

在给定 \$Y_{k-1}, U_{k-1}\$ 的条件下，\$x_k\$ 和 \$y_k\$ 是联合高斯的：

\$$ \\begin{aligned} E[x_k \\mid \\cdot] &= \\hat x_{k|k-1} \\\\ E[y_k \\mid \\cdot] &= C \\hat x_{k|k-1} \\\\ \\text{Cov}[x_k \\mid \\cdot] &= P_{k|k-1} \\\\ \\text{Cov}[y_k \\mid \\cdot] &= C P_{k|k-1} C' + R \\\\ \\text{Cov}[x_k, y_k \\mid \\cdot] &= P_{k|k-1} C' \\end{aligned} \$$

步骤 2：代入条件高斯公式

\$$ \\begin{aligned} \\hat x_{k|k} &= \\hat x_{k|k-1} + P_{k|k-1} C' [C P_{k|k-1} C' + R]^{-1} (y_k - C \\hat x_{k|k-1}) \\\\\\\\ P_{k|k} &= P_{k|k-1} - P_{k|k-1} C' [C P_{k|k-1} C' + R]^{-1} C P_{k|k-1} \\end{aligned} \$$

与几何方法得到的表达式完全一致！

💡 概率法 vs 几何法

概率法自动得到了正确结果，不需要投影定理的几何直觉。它只需要知道"在高斯分布下，条件均值就是 MMSE"。

6. MMSE = LMMSE：线性高斯下的等价性

现在我们终于可以回答核心问题了：

✅ 结论

在线性高斯系统中，MMSE 估计等于 LMMSE 估计。

证明思路

  1. MMSE 定义：\$\\hat x_{k|k}^{\\text{MMSE}} = E[x_k \\mid Y_k, U_{k-1}]\$，即条件均值
  2. 条件高斯公式告诉我们，条件均值是测量的线性函数
  3. LMMSE 定义：在所有的线性估计器中找均方误差最小的那个
  4. 既然 MMSE 本身（条件均值）已经是线性函数，那么它恰好就是 LMMSE 的最优解

📐 几何方法

投影 → LMMSE

只利用了内积空间结构，不需要概率假设

优点：直观、可视化

局限：只得到 LMMSE

📊 概率方法

条件均值 → MMSE

利用了高斯分布的性质

优点：证明 MMSE = LMMSE

局限：需要高斯假设

🎯 所以结论是

在线性高斯情况下，\$$\\hat x_{k|k}^{\\text{LMMSE}} = \\hat x_{k|k}^{\\text{MMSE}}\$$两者完美重合。这在非高斯系统中一般不成立——非高斯条件下 MMSE 通常严格小于 LMMSE（因为 MMSE 在所有估计器中最优，而 LMMSE 只在线性估计器中最优）。这就是为什么 KF 在高斯假设下如此强大的原因。

7. 分离原理的含义

从上面的推导中，我们还发现了一个重要的性质——分离原理（Separation Principle）：

🔄 分离原理

在线性高斯系统中，最优控制器的设计可以分为两个独立的任务：

  1. 估计器设计：设计 KF 估计状态（不依赖于控制目标）
  2. 控制器设计：假设状态完全已知，设计反馈控制器（不依赖于噪声统计）

两者结合即为最优：\$u_k = -F \\hat x_{k|k}\$

为什么控制输入不影响协方差？因为在给定 \$U_{k-1}\$ 条件下，\$B u_{k-1}\$ 是确定性的确定量——它不引入随机性，不影响误差和协方差。

🧠 估计器

KF 处理不确定性，提供最优状态估计 \$\\hat x_{k|k}\$

🎮 控制器

LQR 等控制器假定状态已知，设计最优反馈 \$u_k = -F x_k\$

结合时，将估计值代入控制器：\$u_k = -F \\hat x_{k|k}\$。这就是 LQG 控制（Linear Quadratic Gaussian control）的核心结构。

📝 自测题

问题 1

为什么在线性高斯系统中 MMSE = LMMSE？

点击查看答案

在线性高斯系统中，给定测量后的条件均值（MMSE）是测量的线性函数（由条件高斯公式保证）。而 LMMSE 是所有线性估计器中 MSE 最小的。既然 MMSE 本身也是线性的，它必然就是那个最优的线性估计器。

问题 2

控制输入 \$B u_k\$ 对 KF 的协方差 \$P_{k|k}\$ 有影响吗？为什么？

点击查看答案

没有影响。控制输入 \$B u_k\$ 在给定 \$U_{k-1}\$ 条件下是确定量，它只影响预测均值，不影响误差和协方差。这是 Separation Principle 的基础——控制律设计和估计器设计可以独立进行。
