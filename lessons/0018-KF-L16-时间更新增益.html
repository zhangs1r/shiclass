1. 回顾：KF 完整五等式

上节课我们从几何投影角度完成了测量更新的五个等式证明。现在回顾完整的 KF 递归方程组：

⏱ Time Update（预测步）

\\[ \\hat x_{k+1|k} = A \\hat x_{k|k} \\] \\[ P_{k+1|k} = A P_{k|k} A' + Q \\]

📏 Measurement Update（更新步）

\\[ e_k = y_k - C \\hat x_{k|k-1} \\] \\[ K_k = P_{k|k-1} C' [C P_{k|k-1} C' + R]^{-1} \\] \\[ \\hat x_{k|k} = \\hat x_{k|k-1} + K_k e_k \\] \\[ P_{k|k} = P_{k|k-1} - K_k C P_{k|k-1} \\]

上节课我们证明了测量更新的五个等式。本讲先完成 Time Update 的证明（等式⑥ - ⑦），然后讨论其性质。

2. 时间更新推导（⑥ - ⑦）

定理：给定 \\(\\hat x_{k|k}\\) 和 \\(P_{k|k}\\)，时间更新为：

\\[ \\boxed{\\hat x_{k+1|k} = A \\hat x_{k|k}} \\qquad \\boxed{P_{k+1|k} = A P_{k|k} A' + Q} \\]

证明：预测均值 \\(\\hat x_{k+1|k}\\)

\\[ \\begin{aligned} \\hat x_{k+1|k} &= \\text{proj}\\{x_{k+1} \\mid \\mathcal{L}_k\\} \\\\ &= \\text{proj}\\{A x_k + w_k \\mid \\mathcal{L}_k\\} \\\\ &= \\text{proj}\\{A x_k \\mid \\mathcal{L}_k\\} + \\text{proj}\\{w_k \\mid \\mathcal{L}_k\\} \\\\ &= A \\cdot \\text{proj}\\{x_k \\mid \\mathcal{L}_k\\} + 0 \\\\ &= A \\hat x_{k|k} \\end{aligned} \\]

其中 \\(\\text{proj}\\{w_k \\mid \\mathcal{L}_k\\} = 0\\) 因为 \\(w_k\\) 与所有历史测量不相关——\\(w_k\\) 只影响未来的状态，不影响当前及过去的测量。

证明：预测误差协方差 \\(P_{k+1|k}\\)

\\[ \\begin{aligned} P_{k+1|k} &= \\langle x_{k+1} - \\hat x_{k+1|k},\\; x_{k+1} - \\hat x_{k+1|k} \\rangle \\\\ &= \\langle A x_k + w_k - A \\hat x_{k|k},\\; A x_k + w_k - A \\hat x_{k|k} \\rangle \\\\ &= \\langle A \\tilde x_{k|k} + w_k,\\; A \\tilde x_{k|k} + w_k \\rangle \\\\ &= A \\underbrace{\\langle \\tilde x_{k|k}, \\tilde x_{k|k} \\rangle}_{P_{k|k}} A' + \\underbrace{\\langle w_k, w_k \\rangle}_{Q} \\\\ &\\quad + \\underbrace{A \\langle \\tilde x_{k|k}, w_k \\rangle}_{=0} + \\underbrace{\\langle w_k, \\tilde x_{k|k} \\rangle A'}_{=0} \\\\ &= A P_{k|k} A' + Q \\end{aligned} \\]

交叉项为零是因为 \\(w_k \\perp \\mathcal{L}_k\\)，而 \\(\\tilde x_{k|k} \\in \\mathcal{L}_k\\)。

💡 直观理解

时间更新就是在没有新测量的情况下，用系统模型"推进一步"：均值通过 \\(A\\) 传播，不确定性通过 \\(A\\) 传播后还要加上过程噪声 \\(Q\\)。

3. 预测与估计的完整递归步

现在我们可以把预测和估计之间的关系写成完整的递归形式。给定 \\(\\hat x_{k-1|k-1}\\) 和 \\(P_{k-1|k-1}\\)：

预测步

\\[ \\begin{aligned} \\hat x_{k|k-1} &= A \\hat x_{k-1|k-1} \\\\ P_{k|k-1} &= A P_{k-1|k-1} A' + Q \\end{aligned} \\]

更新步

\\[ \\begin{aligned} K_k &= P_{k|k-1} C' [C P_{k|k-1} C' + R]^{-1} \\\\ \\hat x_{k|k} &= \\hat x_{k|k-1} + K_k (y_k - C \\hat x_{k|k-1}) \\\\ P_{k|k} &= P_{k|k-1} - K_k C P_{k|k-1} \\end{aligned} \\]

🔄 递归节奏

预测 → 测量到来 → 更新 → 再预测 → 这是一个永不停息的循环。每来一个新的测量，只需要常数时间的计算。

4. 协方差的可离线计算特性

观察 KF 的递归等式，可以发现一个重要性质：

\\[ \\begin{aligned} P_{k|k-1} &\\;\\text{only depends on}\\; P_{k-1|k-1} \\\\ K_k &\\;\\text{only depends on}\\; P_{k|k-1} \\\\ P_{k|k} &\\;\\text{only depends on}\\; P_{k|k-1} \\\\ \\end{aligned} \\]

特别地，\\(P_{k|k}\\) 和 \\(K_k\\) 完全不依赖测量值 \\(y_k\\)！它们只由系统参数 \\((A, C, Q, R)\\) 和初始协方差 \\(\\Pi_0\\) 决定。

🚀 实际意义

这意味着我们可以离线（offline）预先计算所有的 \\(P_{k|k}\\) 和 \\(K_k\\)，存成表格。在线运行时只需要查表 + 简单的向量运算即可更新状态估计。

这在实际工程中是巨大的优势。比如嵌入式系统计算资源有限，提前算好增益序列就能极大降低在线负担。

5. \\(P_{k|k}\\) 的收敛性

📊 模拟实验显示：\\(P_{k|k}\\) 随 \\(k\\) 增加而趋于一个稳态值 \\(\\bar P\\)

问题

随着时间 \\(k \\to \\infty\\)，\\(P_{k|k}\\) 是否会收敛到一个稳态值？

答案

是的，当以下两个条件满足时，\\(P_{k|k}\\) 收敛：

📌 可观性

\\[ (A, C)\\; \\text{observable} \\]

系统的所有模态都能通过测量被观测到

📌 能控性

\\[ (A, \\sqrt{Q})\\; \\text{controllable} \\]

过程噪声能激励系统的所有模态

当这两个条件满足时，\\(P_{k|k} \\to \\bar P\\)（一个常数矩阵）。反之，如果某些模态不可观或不可控，这些模态的不确定性可能永远不会降低或收敛。

🎯 重要意义

收敛意味着 \\(K_k \\to K^*\\)（一个常数矩阵），这进一步简化了在线计算——我们甚至不需要查表，只需要记住一个固定的增益矩阵。

6. 固定增益估计器

当 \\(P_{k|k}\\) 收敛到 \\(\\bar P\\) 后，Kalman 增益也收敛到一个常数：

\\[ K_k \\longrightarrow K^* \\]

此时估计器退化为一个简单的线性时不变系统：

\\[ \\boxed{\\hat x_{k|k} = A \\hat x_{k-1|k-1} + K^* [y_k - C A \\hat x_{k-1|k-1}]} \\]

或者等价地：

\\[ \\boxed{\\hat x_{k+1|k} = A \\hat x_{k|k-1} + K^* [y_k - C \\hat x_{k|k-1}]} \\]

固定增益估计器的优势

⬇️ 计算量显著降低

每一步只需：
① 矩阵-向量乘法 \\(A \\hat x\\)
② 向量加减
③ 常数矩阵-向量乘法 \\(K^*\\)

不再需要在线求逆或矩阵乘法。

💾 存储空间降为 \\(O(1)\\)

只需要存储一个常数增益矩阵 \\(K^*\\)，无需存储整个增益序列。

对于运行时间极长的系统（如卫星导航），这是巨大的节约。

🧠 思考

当然，\\(P_{k|k}\\) 不会严格收敛到绝对的零——不确定性永远不会完全消失。但收敛到一个常数意味着"信息收益"与"信息损失"达到了动态平衡。

7. 传感器融合

在实际系统中，我们往往有多个传感器。每个传感器提供不同的测量信息。Kalman Filter 提供了一种优雅的传感器融合（Sensor Fusion）方案。

问题设定

假设有 \\(n\\) 个传感器，第 \\(i\\) 个传感器的测量模型为：

\\[ y_k^i = C_i x_k + v_k^i, \\quad v_k^i \\sim N(0, R_i) \\]

我们可以将所有测量堆叠在一起：

\\[ y_k = \\begin{bmatrix} y_k^1 \\\\ y_k^2 \\\\ \\vdots \\\\ y_k^n \\end{bmatrix} = C x_k + v_k, \\quad C = \\begin{bmatrix} C_1 \\\\ C_2 \\\\ \\vdots \\\\ C_n \\end{bmatrix}, \\quad v_k \\sim N(0, R) \\]

然后直接套用标准的 KF 测量更新。

Plug-and-Play 特性

将测量方程展开后，KF 测量更新可以写成每个传感器贡献之和的形式：

\\[ \\begin{aligned} P_{k|k}^{-1} &= P_{k|k-1}^{-1} + \\sum_{i=1}^{n} C_i' R_i^{-1} C_i \\\\ P_{k|k}^{-1} \\hat x_{k|k} &= P_{k|k-1}^{-1} \\hat x_{k|k-1} + \\sum_{i=1}^{n} C_i' R_i^{-1} y_k^i \\end{aligned} \\]

这就是信息形式（Information Form）的 KF。它的优势在于：

  • 模块化：每个传感器的信息贡献独立，互不干扰
  • 即插即用：某个传感器的数据来了就加进去，没来就不加
  • 容错性强：某个传感器失效时，只需移除其贡献项

🔗 应用场景

自动驾驶汽车有摄像头、激光雷达、IMU、GPS 等多种传感器，每个传感器可能随时掉线或信号延迟。信息形式的 KF 天然支持这种动态配置的融合需求。

8. 从几何到概率的过渡

到目前为止，我们一直使用几何方法（投影定理 + 新息正交性）来推导 KF。这种方法的优点是直观，但有一个问题没有解决：

🤔 开放问题

我们得到的只是线性最优估计（LMMSE）。在线性高斯系统下，它到底是不是全局最优（MMSE）？

要回答这个问题，我们需要从概率的角度重新审视估计问题。这就是下一讲的内容——用条件高斯分布推导 MMSE 估计，并证明在线性高斯系统中 MMSE = LMMSE。

同时，下一讲还会引入控制输入（\\(B u_k\\)），并介绍第三种推导 KF 的方法——优化法（将 \\(P_k\\) 写成 \\(K_k\\) 的二次型然后最小化）。

📝 自测题

问题 1

为什么 KF 的协方差 \\(P_{k|k}\\) 和增益 \\(K_k\\) 与测量值 \\(y_k\\) 无关？

点击查看答案

因为 \\(P_{k|k}\\) 的更新只依赖于系统参数 \\((A, C, Q, R)\\) 和上一步的 \\(P_{k-1|k-1}\\)，不包含任何测量值。测量值只影响状态估计 \\(\\hat x_{k|k}\\) 的轨迹，不影响不确定性。

问题 2

\\(P_{k|k}\\) 收敛的条件是什么？固定增益估计器有什么好处？

点击查看答案

条件：(A, C) observable 且 (A, √Q) controllable。好处：(1) 计算量显著降低——只需简单的矩阵-向量运算；(2) 存储空间降为 O(1)——只需存一个常数增益矩阵。
