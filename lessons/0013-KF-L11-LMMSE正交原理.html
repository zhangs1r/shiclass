1. 线性估计器

【课堂例题】 估计房间温度 \(T\)，有两个测量 \(y_1, y_2\)：

\[ \hat T = k_1 y_1 + k_2 y_2 \]

矩阵形式：

\[ \hat x = K Y \]

📌 注意

非线性估计器例如 \(\hat T = \sin(y_1) + \sin(y_2)\)。LMMSE 只考虑线性形式，因为线性估计便于分析和计算。

2. 误差和误差协方差

估计误差：

\[ e = x - \hat x = x - K Y \]

误差协方差：

\[ P(K) = E[ee^T] \]

LMMSE 要找 \(K^*\)，使：

\[ P(K^*) \le P(K),\quad \forall K \]

这里的 \(\le\) 是半正定意义（矩阵比较）。

3. 为什么误差协方差小意味着 MSE 小

均方误差：

\[ E[e^Te] = \operatorname{Tr}(E[ee^T]) \]

所以如果误差协方差矩阵更小（半正定意义），它的迹（trace）也更小，即 MSE 更小。

4. 零均值假设

课堂暂时假设所有随机变量零均值。定义协方差矩阵：

\(R_x\)

\[ R_x = E[xx^T] \]

\(R_y\)

\[ R_y = E[yy^T] \]

\(R_{xy}\)

\[ R_{xy} = E[xy^T] \]

💡 非零均值处理

如果变量非零均值，可以先中心化（减去均值），估计后再加回均值。这样做不影响协方差结构。

5. LMMSE 正规方程

核心结论：

\[ K^* R_y = R_{xy} \]

若 \(R_y\) 可逆：

\[ K^* = R_{xy}R_y^{-1} \]

这就是 LMMSE 的核心公式，也是贯穿整个 Kalman Filter 理论的基础。

6. 推导直觉

把误差协方差展开：

\[ P(K) = E[(x - KY)(x - KY)^T] \]

展开后会得到一个关于 \(K\) 的二次型。像第 06 讲（二次型优化）一样配方，最小点满足：

\[ K R_y = R_{xy} \]

这本质上是一个无约束二次优化问题，梯度为零给出正规方程。

7. 正交原理

从正规方程出发：

\[ K R_y = R_{xy} \;\Longrightarrow\; K E[YY^T] = E[XY^T] \]

移项：

\[ E[(X - KY)Y^T] = 0 \]

即：

\[ E[(X - \hat X) Y^T] = 0 \]

这就是正交原理：最优估计的误差与观测不相关。

8. 正交原理的直觉

🧠 核心直觉

最优估计之后，误差不应该再和测量有线性相关。

如果误差还和 \(Y\) 有关，说明 \(Y\) 里还有信息没用完，可以继续改进估计。

所以：最优线性估计的剩余误差，是所有线性测量信息都解释不了的部分。

9. 【补充例题】标量 LMMSE

设 \(x, y\) 是零均值标量：

\[ E[y^2] = 4,\quad E[xy] = 2 \]

线性估计 \(\hat x = k y\)，则：

\[ k^* = \frac{E[xy]}{E[y^2]} = \frac{2}{4} = 0.5 \]

所以：

\[ \hat x = 0.5\,y \]

如果 y 有 4 个单位的"功率"，x 和 y 共享 2 个单位的相关性，最优增益就是 0.5。

10. 最优误差协方差

若 \(R_y\) 可逆：

\[ P(K^*) = R_x - R_{xy}R_y^{-1}R_{yx} \]

📖 解释

原始不确定性 \(R_x\) 减去测量解释掉的部分 \(R_{xy}R_y^{-1}R_{yx}\)。测量越有用，这部分越大，最终误差越小。

11. 和 Kalman Filter 的关系

Kalman Filter 的增益有形式：

\[ K_k = P_{k|k-1}C^T(CP_{k|k-1}C^T + R)^{-1} \]

这其实就是 \(R_{xy}R_y^{-1}\) 在特定模型下的结果：

  • \(P_{k|k-1}C^T\) 对应 \(R_{xy}\)（状态预测误差与测量的互协方差）
  • \(CP_{k|k-1}C^T + R\) 对应 \(R_y\)（测量协方差）

📝 自测题

问题 1

为什么 LMMSE 的误差与测量正交？

点击查看答案

这是最优性的一阶必要条件。如果误差与测量相关，说明还可以通过调整 K 来减小 MSE。正交条件 \(E[(X-\hat X)Y^T]=0\) 等价于梯度为零。

问题 2

\(R_y\) 不可逆时会发生什么？

点击查看答案

如果 \(R_y\) 不可逆，说明测量之间线性相关（有冗余），或者某个测量方向方差为零。此时 \(R_y^{-1}\) 不存在，但正规方程 \(K R_y = R_{xy}\) 仍然有解（多解），可以用伪逆处理。

如果测量噪声很大，直觉上 Kalman gain \(K\) 应该大还是小？
